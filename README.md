# FalconERP - Sistema ERP Simple

Sistema ERP simple para facturación y gestión de productos, clientes y empresas.

## Características

- ✅ Gestión de empresas
- ✅ Gestión de productos
- ✅ Gestión de clientes
- ✅ Facturación
- ✅ Autenticación de usuarios
- ✅ Multi-empresa

## Tecnologías

- Next.js 14
- TypeScript
- Prisma ORM
- PostgreSQL
- TailwindCSS
